# Task 2 – Reflection Notes

## Platforms I Chose: GitHub, Kaggle, and Stack Overflow

**GitHub** is a platform where developers store and share code. I created a profile README with my name, branch, year, and what I hope to learn. It will act as my live portfolio throughout college.

**Kaggle** is focused on data science and machine learning. It has datasets, notebooks, and competitions. I created an account to start exploring this field early, even as a beginner.

**Stack Overflow** is a Q&A community for programmers. Whenever I get stuck on a coding problem, I can search for solutions here. Over time, I hope to also answer questions and help others.

## How I Plan to Use Them Over the Next Four Years

- **GitHub** – Store all assignments and projects, share profile with recruiters
- **Kaggle** – Attempt beginner competitions once I learn Python and statistics
- **Stack Overflow** – Use daily for doubt solving while learning to code

These three platforms together build the foundation of my professional online presence as a CS student.
